# Tech Stack - ClueScale

## Frontend
- **Framework:** React (v19)
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** 
    - Tailwind CSS (Utility-first styling)
    - Custom CSS (for gradient and sketchbook themes)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Icons/Visuals:** SVG Doodles (Sketchbook theme)

## Backend
- **Runtime:** Node.js
- **Language:** TypeScript
- **Web Framework:** Express
- **Real-time Engine:** Socket.IO
- **HTTP Client:** Axios (for integration with GameBuddies API)

## Infrastructure & Integration
- **Platform Integration:** GameBuddies.io (Lobby, Player, and Return APIs)
- **Environment Management:** Dotenv
- **Deployment:** Render.yaml (Configured for easy deployment)
- **Local Development:** Nodemon and Concurrently (for running client and server)

## Core Technologies
- **MediaPipe:** Vision tasks (likely for mascot or avatar features)
- **Three.js:** 3D rendering (referenced for avatar/model support)
- **Emoji Picker:** Integration for social expression
